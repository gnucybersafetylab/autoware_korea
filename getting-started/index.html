시작하기

Autoware와 함께 자율주행 여정을 시작하세요.

  GitHub에서 시작

Autoware 프로젝트는 GitHub에서 오픈소스로 제공됩니다. 코드를 다운로드하고 기여할 수 있습니다.

주요 저장소:

  • Autoware Universe (최신 버전)
  • Autoware.Auto (안정화 버전)
  • 문서 및 튜토리얼

GitHub 방문

 워크그룹 참여

Autoware Foundation의 다양한 워크그룹에 참여하여 프로젝트 개발에 기여하세요.

주요 워크그룹:

  • Perception & Localization WG
  • Planning & Control WG
  • Simulation WG
  • Hardware WG

커뮤니티 보기

 문서 및 튜토리얼

Autoware의 공식 문서와 튜토리얼을 통해 자율주행 시스템 구축 방법을 학습하세요.

학습 자료:

  • 설치 가이드
  • 시뮬레이션 튜토리얼
  • API 문서
  • 베스트 프랙티스

문서 보기

 Discord 커뮤니티

Discord 서버에 참여하여 다른 개발자들과 소통하고 질문을 나누세요.

실시간으로 개발자들과 소통하고, 기술적인 질문을 하거나 최신 소식을 받아보세요.

Discord 참여

 Open AD Kit

Open AD Kit을 통해 자율주행 개발을 위한 통합 솔루션을 활용하세요.

하드웨어와 소프트웨어를 통합한 개발 키트로 빠르게 프로토타입을 구축할 수 있습니다.

자세히 보기

 멤버십

기업이나 기관으로서 Autoware Foundation의 멤버가 되어 전략적 방향에 영향을 미치세요.

멤버십을 통해 프로젝트의 전략적 방향에 참여하고, 네트워킹 기회를 얻을 수 있습니다.

설치 가이드

시스템 요구사항

  • 운영체제: Ubuntu 22.04 LTS (권장)
  • ROS: ROS 2 Humble 또는 Foxy
  • 하드웨어: NVIDIA GPU (권장), 최소 16GB RAM
  • 디스크 공간: 최소 50GB 여유 공간

빠른 시작

# ADE(Agile Development Environment) 사용
$ git clone https://github.com/autowarefoundation/autoware.git
$ cd autoware
$ ansible-playbook setup-dev-env.yml
$ colcon build --symlink-install
